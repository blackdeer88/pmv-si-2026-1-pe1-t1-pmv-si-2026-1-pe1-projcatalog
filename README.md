# Catalog

`CURSO: Sistemas de Informação`

`DISCIPLINA: Projeto - Aplicações Web`

`SEMESTRE: 1º`

O projeto consiste no desenvolvimento de uma plataforma digital voltada para a divulgação de pequenos negócios locais. A ideia surgiu a partir da dificuldade que muitos empreendedores têm para divulgar seus produtos e serviços na internet, além da dificuldade dos consumidores em encontrar opções próximas de forma rápida e organizada.A plataforma permitirá que empreendedores se cadastrem e divulguem seus negócios, adicionando produtos ou serviços de forma simples. Já os usuários poderão pesquisar diretamente pelo que procuram, encontrando estabelecimentos próximos que atendam à sua necessidade. Dessa forma, o sistema busca facilitar a conexão entre clientes e pequenos negócios, promovendo maior visibilidade e acesso aos serviços locais.

## 1. Documentação de Contexto

### Problema
Muitos pequenos negócios e trabalhadores autônomos não possuem conhecimento técnico ou recursos financeiros para investir em divulgação online. Como consequência, possuem baixa visibilidade e os consumidores encontram dificuldades para localizar produtos e serviços na região, uma vez que as informações estão dispersas.

### Objetivos

#### Objetivo Geral
**Desenvolver** a plataforma **Catalog**, uma solução móvel de conexão hiperlocal que combina geolocalização, curadoria visual por *swipe* e um fluxo de contato simplificado para fortalecer a visibilidade de empreendedores regionais.

#### Objetivos Específicos
* **Criar** um sistema de busca inteligente por proximidade e categorias em tempo real.
* **Projetar** uma interface baseada em *cards* visuais para favoritar ("Agenda") ou ocultar ("Hoje Não") estabelecimentos.
* **Desenvolver** um módulo de autogestão para que o empreendedor cadastre seus serviços e link direto de WhatsApp.
* **Integrar** um sistema de avaliações e comentários para validar a qualidade dos serviços perante a comunidade.

### Justificativa
76% dos empreendedores enfrentam dificuldades com estratégias de marketing (Instituto Locomotiva). O **Catalog** atua como uma ferramenta acessível para ampliar a visibilidade desse público utilizando práticas de mercado adaptadas a recursos limitados.
